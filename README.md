## 🚀 Quick Start

### Prerequisites
```
✅ Node.js 16+
✅ Python 3.9+
✅ Groq API Key (free at console.groq.com)
```

### Installation (3 Steps)

```bash
# 1️⃣ Clone the repository
git clone https://github.com/anshxgaur/MODEL-X.git
cd MODEL-X

# 2️⃣ Install dependencies
pip install -r requirements.txt
npm install

# 3️⃣ Set up environment variables
echo "GROQ_API_KEY=your_api_key_here" > .env
echo "VITE_GROQ_API_KEY=your_api_key_here" >> .env

# 4️⃣ Run NOVA
python app.py          # Terminal 1: Start Flask backend
npm run dev            # Terminal 2: Start React frontend
```

**✅ NOVA is now running at `http://localhost:5173`**

## 🎓 Key Technologies Explained

### Groq LPU - The Secret Sauce 🔧
- **Language Processing Unit**: Specialized hardware optimized for AI inference
- **Speed**: 750+ tokens/second (10x faster than traditional GPUs)
- **Efficiency**: Lower latency, reduced costs, faster iterations
- **Models**: Llama 3.3 70B, Mixtral 8x7B, and more

### React + TypeScript - Type-Safe UI 🎨
- **Component-Based**: Modular, reusable UI components
- **Type Safety**: Catch errors at compile time, not runtime
- **Performance**: Virtual DOM with efficient re-rendering
- **Developer Experience**: Hot Module Replacement (HMR) for instant feedback

### Flask Backend - Robust API 🔌
- **REST API**: Simple HTTP endpoints for communication
- **Async Processing**: Handle multiple requests concurrently
- **Middleware**: CORS support for secure cross-origin requests
- **Extensibility**: Easy to add new endpoints and features

## 🚀 Advanced Features

### 1. **Intelligent Caching**
- First request: Query Groq API
- Subsequent identical requests: Instant cache hit
- Smart normalization: Handles typos and variations
- **Result**: 90% faster responses for common queries

### 2. **Multi-Modal AI Router**
```python
# Automatic fallback strategy:
1. Check Local Cache         → Instant (0ms)
```
